# Project-lora-pager-esp32
# 📟 Пейджер на базе LoRa и ESP32

Проект устройства для приёма сообщений с передачей данных по радиоканалу.

---

## 🎯 Описание проекта

Устройство представляет собой пейджер, использующий модуль **LoRa** для беспроводной передачи данных и микроконтроллер **ESP32** для обработки сигналов.

---

## 🛠 Технические характеристики

| Параметр | Значение |
|----------|----------|
| **Микроконтроллер** | ESP32 |
| **Радиомодуль** | LoRa (433 MHz / 868 MHz) |
| **Печатная плата** | 2-х слойная |
| **САПР** | Altium Designer |
| **Статус** | В разработке |

---

## 📐 Этапы разработки

- ✅ Проектирование архитектуры устройства
- ✅ Подбор компонентной базы
- ✅ Разработка 2-х слойной печатной платы в Altium Designer
- ✅ Трассировка
- ✅ Командное взаимодействие от идеи до прототипирования
